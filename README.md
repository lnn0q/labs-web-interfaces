# Назва проекту: Takibi - Платформа для спільнот та фандомів

**Виконав:** Денисенко Богдан Романович  
**Група:** КВ-51мн

## Про проект

Takibi — це сучасна українська соціальна платформа, розроблена спеціально для гіків, анімешників та фанатів різних всесвітів. Назва походить від японського слова "вогнище" (焚き火) і символізує затишок спілкування серед однодумців. Додаток дозволяє об'єднувати людей у фандоми за інтересами, створювати публікації та спілкуватися в чатах у реальному часі.

### Основна функціональність:

1. **Каталог фандомів**: Перегляд, пошук та фільтрація доступних спільнот.
2. **Автентифікація**: Реєстрація та вхід користувачів із використанням JWT-токенів.
3. **Сторінка Фандому**: Обговорення (стрічка постів, коментарі, лайки) та реалтайм-чат для учасників.
4. **Особистий кабінет**: Керування профілем та перегляд своїх фандомів.
5. **Адміністративна панель**: Керування користувачами, створення нових фандомів, модерація.
6. **Запити на фандоми**: Користувачі можуть подавати заявки на створення нових тематичних спільнот.

### Використані технології:

- **Клієнтська частина (Web)**: ReactJS, Redux Toolkit, React Router v6.
- **Стилізація**: Material-UI з адаптивним дизайном.
- **Мобільний додаток**: React Native, React Navigation, AsyncStorage.
- **Взаємодія з сервером**: Axios (REST API).
- **Збереження стану**: Redux Store / AsyncStorage для токенів доступу.
- **Бекенд**: Django REST Framework, PostgreSQL, Redis, Celery.
- **Документація API**: drf-spectacular (OpenAPI 3.0, Swagger UI, ReDoc).
- **Інфраструктура**: Docker, Docker Compose, Nginx.
