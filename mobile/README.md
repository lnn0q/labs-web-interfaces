## Опис додатку

Takibi Mobile - мобільний клієнт для платформи Takibi, побудований на React Native (Expo). Додаток підключається до серверного REST API (Django REST Framework) та надає наступну функціональність:

### Функціональність:

1. **Аутентифікація** — вхід до системи за допомогою email та паролю (JWT-токени).
2. **Реєстрація** — створення нового акаунту (ім'я, email, пароль, стать, дата народження).
3. **Каталог фандомів** — перегляд та фільтрація спільнот за категоріями.
4. **Сторінка фандому** — перегляд постів, створення публікацій, коментарі, лайки.
5. **Чат** — обмін повідомленнями в реальному часі (WebSocket).
6. **Профіль** — перегляд та редагування даних, перегляд своїх фандомів.

### Використані технології:

- React Native (Expo)
- React Navigation
- Axios (REST API)
- AsyncStorage (збереження токенів)
